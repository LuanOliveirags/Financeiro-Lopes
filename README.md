# Controle de Finanças Pessoais - PWA

Este é um app PWA para controle de finanças pessoais e conjuntas.

## Como instalar como PWA
1. Abra o site em um navegador compatível (Chrome, Edge, etc.).
2. Clique no botão "Instalar" na barra de endereço ou menu.
3. O app será instalado como um aplicativo nativo.

## Funcionalidades
- Adicionar transações, dívidas e salários.
- Dashboard com gráficos.
- Importar dados do Excel.
- Tema claro/escuro.
- Funciona offline (cache básico).

## Ícones
Para personalizar, adicione ícones em `img/`:
- `icon-192.png` (192x192)
- `icon-512.png` (512x512)

Use ferramentas como Canva ou online para criar ícones com tema financeiro (moeda, gráfico, etc.).

## Desenvolvimento
Para testar localmente, use um servidor HTTP (ex: `python -m http.server`).
